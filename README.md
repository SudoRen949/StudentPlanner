# Student AI Study Planner - Frontend (React + Vite)

A modern, mobile-responsive web application built with React, Vite, and Tailwind CSS. This application serves as the user interface for students to manage assignments, view smart notifications, chat with an advanced AI study coach, and generate optimized daily study schedules.

## Tech Stack

- **Framework:** React 18 (TypeScript)
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **HTTP Client:** Axios (for API communication)
- **State & Auth:** Custom React Context Provider (`AuthProvider`)

## Features Implemented

- **Smart Dashboard:** Overview of daily progress and quick navigation.
- **AI Schedule Generator:** Fetches student context and displays a chronologically arranged daily study plan generated by AI.
- **Completed Tasks:** Tracks accomplishments by pulling assignments marked `completed: true` with dynamic visual treatments (strike-through cards and gradients).
- **Proximity Notifications:** Calculates real-time countdown deadlines locally and displays critical/warning alerts if an assignment is due within 1–3 days.
- **AI Chat Coach:** Context-aware, scroll-locked conversational interface powered by open-source LLMs.
- **Student Profile:** Simulated academic ID card displaying school, course, and year variables alongside an inline Change Password modal popup.
